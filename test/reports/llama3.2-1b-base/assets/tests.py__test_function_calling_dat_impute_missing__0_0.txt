**Impute Missing Values in Age Column using Median Strategy**

We will use the `impute_missing` function from the `clean_dataset` module to fill missing values in the `age` column. We will also use the `detect_outliers` function to identify outliers in the data.

```json
{
    "function": "impute_missing",
    "column": "age",
    "strategy": "median"
}
```

**JSON Response**

```json
{
    "function": "impute_missing",
    "column": "age",
    "strategy": "median",
    "operations": [
        {
            "function": "detect_outliers",
            "column": "age",
            "method": "quantile"
        }
    ]
}
```

**Explanation**

1. We start by specifying the `impute_missing` function, which is used to fill missing values.
2. We specify the `column` as `age`, which is the column we want to impute missing values in.
3. We specify the `strategy` as `median`, which is the method we want to use to impute missing values.
4. We also specify the `operations` list, which contains the `detect_outliers` function. This function identifies outliers in the data.
5. Finally, we specify the `column` and `method` as `quantile`, which is the method we want to use to identify outliers.

**Example Use Case**

Suppose we have a dataset with missing values in the `age` column. We want to impute these missing values using the median strategy and identify outliers in the data.

```json
{
    "function": "clean_dataset",
    "dataset": "data.json",
    "operations": [
        {
            "function": "impute_missing",
            "column": "age",
            "strategy": "median"
        },
        {
            "function": "detect_outliers",
            "column": "age",
            "method": "quantile"
        }
    ]
}
```

**JSON Response**

```json
{
    "function": "clean_dataset",
    "dataset": "data.json",
    "operations": [
        {
            "function": "impute_missing",
            "column": "age",
            "strategy": "median"
        },
        {
            "function": "detect_outliers",
            "column": "age",
            "method": "quantile"
        }
    ]
}
```

This will return the cleaned dataset with imputed missing values and outliers identified using the median strategy.
