Here's a JSON configuration for the functions you've mentioned:

```json
{
  "function": "configure_network",
  "vpc_id": "vpc-12345",
  "subnet": "10.0.1.0/24",
  "internet_gateway": true
}
```

This configuration will:

- Configure a VPC with ID `vpc-12345`
- Create a subnet with IP address `10.0.1.0/24`
- Enable the Internet Gateway for the VPC.
