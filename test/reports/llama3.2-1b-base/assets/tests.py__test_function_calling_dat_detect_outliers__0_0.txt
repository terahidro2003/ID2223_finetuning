Here's an example of how you can use the IQR (Interquartile Range) method to detect outliers in the price column of a dataset.

```python
import pandas as pd
import numpy as np

def detect_outliers(price_column, method='IQR'):
    """
    Detect outliers in a column of a dataset using the IQR method.

    Parameters:
    price_column (str): Name of the column containing the prices.
    method (str, optional): Method to use for detecting outliers. Defaults to 'IQR'.
    """

    # Ensure the column is numeric
    price_column = pd.to_numeric(price_column, errors='coerce')

    # Calculate the first quartile (Q1) and third quartile (Q3)
    Q1 = np.percentile(price_column, 25)
    Q3 = np.percentile(price_column, 75)

    # Calculate the interquartile range (IQR)
    IQR = Q3 - Q1

    # Define the lower and upper bounds for outliers
    lower_bound = Q1 - 1.5 * IQR
    upper_bound = Q3 + 1.5 * IQR

    # Detect outliers
    outliers = price_column[(price_column < lower_bound) | (price_column > upper_bound)]

    return outliers

# Example usage
price_data = pd.DataFrame({
    'Price': [100, 120, 110, 130, 140, 160, 170, 180]
})

outliers = detect_outliers(price_data['Price'], method='IQR')

print("Outliers:")
print(outliers)
```

In this code:

1. We define a function `detect_outliers` that takes a column name and an optional method ('IQR' or 'mean') as parameters.
2. We convert the column to numeric type using `pd.to_numeric`.
3. We calculate the first quartile (Q1) and third quartile (Q3) using `np.percentile`.
4. We calculate the interquartile range (IQR) using `Q3 - Q1`.
5. We define the lower and upper bounds for outliers using `Q1 - 1.5 * IQR` and `Q3 + 1.5 * IQR`, respectively.
6. We detect outliers by checking if the values are less than the lower bound or greater than the upper bound.
7. We provide an example usage of the function with a sample dataset.

This code will output the outliers in the price column.
