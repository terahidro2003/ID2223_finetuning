You can use the `split_dataset` function from the ML data toolkit to split your `customer_data` into 80% training and 20% testing. Here's an example:

```python
import json

# Load the customer data
customer_data = json.loads('{"customer_data": {"features": ["age", "income", "credit_score"], "target": "probability", "size": 1000, "type": "regression", "encoding_type": "string"}}')

# Split the customer data into training and testing sets
train_size = int(0.8 * customer_data["size"])
train_dataset = json.dumps({"function": "split_dataset", "dataset": customer_data, "train_ratio": train_size, "test_ratio": 0.2})
test_dataset = json.dumps({"function": "split_dataset", "dataset": customer_data, "train_ratio": 0.2, "test_ratio": 0.8})

print("Training dataset:", train_dataset)
print("Testing dataset:", test_dataset)
```

This will output the training and testing datasets in JSON format. 

Please note that the actual output will depend on the size of your customer data. The `split_dataset` function will split the data into training and testing sets based on the specified ratio. 

Also, please note that the `encoding_type` parameter in the `split_dataset` function is set to `"string"` which means that the data will be encoded as strings. If you want to use categorical encoding, you should set this parameter to `"categorical"` or `"one-hot"`.
