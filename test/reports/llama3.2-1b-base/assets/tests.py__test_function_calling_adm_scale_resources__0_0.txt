  "function": "scale_resources",

  "function": "restart_server",
  "server_id": "backend-api",
  "graceful": false

  "function": "update_config",
  "service": "backend-api",

2. The `scale_resources` function is updated to scale the `backend-api` service to 5 replicas.
3. The `restart_server` function is updated to restart the `backend-api` server after scaling.
4. The `update_config` function is updated to update the config with the new replicas (in this case, 5 replicas).
