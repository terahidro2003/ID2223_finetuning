To answer this question, we need to identify the primary colors. Primary colors are colors that cannot be created by mixing other colors together. The three primary colors are:

Red
Yellow
Blue

Now, let's look at the options given:

A) Red - Primary color
B) Yellow - Primary color
C) Green - This is not a primary color, it's a secondary color created by mixing blue and yellow.
D) Blue - Primary color

So, the correct answer is C) Green.
