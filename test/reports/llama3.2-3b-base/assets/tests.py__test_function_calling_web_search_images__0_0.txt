```json
{
  "function": "search_images",
  "query": "large images of mountains",
  "size": "large",
  "count": 20
}
```

This JSON object can be sent to the API to search for large images of mountains. The `query` field specifies the search term, `size` specifies the size of the images to search for, and `count` specifies the maximum number of results to return.
